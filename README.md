# HDI-Predictor-ML-Project

## 📌 Project Overview
The Human Development Index (HDI) Predictor is a Machine Learning web application that predicts a country's Human Development Index (HDI) score using Linear Regression. The application is developed using Python, Flask, Scikit-learn, Pandas, NumPy, Matplotlib, and Seaborn.

---

## 🚀 Features
- Predicts Human Development Index (HDI)
- Machine Learning based prediction using Linear Regression
- User-friendly Flask web interface
- Fast and accurate predictions
- Simple and responsive design

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/karnayanaBala/HDI-Predictor-ML-Project.git
```

Go to the project folder

```bash
cd HDI-Predictor-ML-Project
```

Install required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
